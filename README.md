# EasyVisa: Visa Approval Prediction Using Ensemble Machine Learning Models

## Overview
This project analyzes visa applicant data to build predictive machine learning models that support visa certification decision-making. Multiple ensemble learning techniques were evaluated and optimized to identify the key factors influencing visa approval outcomes.

## Business Objective
The objective of this project is to help streamline visa approval analysis by predicting certification outcomes and identifying the most influential applicant and employer-related factors.

## Skills and Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Ensemble Learning
- Hyperparameter Tuning
- Classification Models

## Models Evaluated
- AdaBoost Classifier
- Gradient Boost Classifier
- XGBoost Classifier

## Final Model Selection
The Tuned Gradient Boost Classifier was selected as the final model based on its strong validation F1-score, balanced precision-recall performance, and good generalization across datasets.

## Key Insights
- Education level, prevailing wage, and prior work experience were among the most influential factors affecting visa approval outcomes.
- The final model demonstrated stable performance across training, validation, and test datasets.
- Competitive wages and stronger educational qualifications were associated with higher certification likelihood.
